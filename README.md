# JShint
> A default JSHint configuration file


###Remember to load the file:
On WebStorm, go to WebStorm, Preferences, Search: jshint, Select: Use jshint, Select: use config files and then browse for the .jshintrc file on the root of your project

**This files works well for many projects, if you have some extra properties please let me know.**
